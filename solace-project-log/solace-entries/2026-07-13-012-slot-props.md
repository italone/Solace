## 变动摘要

为 slot 函数新增可选 props 参数。组件现在可以通过 `slots.default?.({ text: "Body" })` 或 `slots.item?.({ value })` 向父级 slot 函数传递数据；现有无参 slot 调用保持兼容。

## 变动原因

Solace 已支持 default slots 和 named slots，但 slot 函数类型仍是零参数，无法表达 scoped slot 数据。本次补齐最小 slot props 能力，让组件可把内部状态、列表项或索引传给 slot 内容。

## 影响范围

- 影响模块：组件 slot 类型、VNode slot children 类型、组件单元测试、package consumer smoke、API 文档、README。
- 影响对象：公开 TypeScript 类型 `Slot`、`SlotProps`、`Slots`、`VNodeSlots`。
- 行为变化：slot 函数可接收一个可选 props 对象；无参调用继续有效。
- 风险等级：低到中；主要是类型扩展，runtime 不需要改 renderer 主流程。
